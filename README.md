# Secure-Login-System
## Description
Develop a secure login web application.
## Features
- User registration
- User login
- Password hashing using bcrypt
- Input validation
- Protection against SQL injection
- Session management
- Logout feature
- Optional Two-Factor Authentication (2FA)

## Technologies Used
- Python
- Flask
- SQLite
- bcrypt

## Expected Outcome
A secure login system with hashed passwords, input validation, session management, and optional 2FA significantly reduces unauthorized access and protects user accounts from common attacks.
